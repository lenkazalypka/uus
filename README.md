# UUS ONLINE — твой путь к творчеству (v2)

Запуск:

```bash
npm install
npm run dev
```

## Хранение

- Курсы / профиль / лайки / покупки: localStorage
- Инициализация из mock JSON при первом запуске

## Supabase авторизация (опционально)

1. Создай проект в Supabase → Authentication включен
2. В корне создай `.env.local`:

```bash
NEXT_PUBLIC_SUPABASE_URL=...
NEXT_PUBLIC_SUPABASE_ANON_KEY=...
```

3. Открой `/auth`.

Важно: Supabase используется только для входа/выхода. Данные платформы остаются в localStorage (как в ТЗ).

## Видео из Яндекс Cloud

- Если у тебя **публичный** URL на mp4 (например из Object Storage) — вставляй его в поле «Видео», на странице курса откроется `<video>`.
- Если объект **приватный** и требует подписи/токена — без серверной выдачи signed URL это не сделать безопасно (секреты нельзя держать на клиенте).
